# Max Messenger Bot

Чек-лист бот для кредитной реструктуризации на платформе MAX Messenger.

## Описание
Бот проводит опрос пользователя по чек-листу и выдает рекомендацию (Зеленая/Желтая/Красная зона).

## Запуск
1. Установите зависимости:
   ```bash
   npm install
   ```
2. Запустите бота:
   ```bash
   npm start
   ```
