# Инструкция по развертыванию (Deploy)

Для того, чтобы бот работал 24/7 без вашего компьютера, его нужно загрузить на хостинг.
Самый простой вариант для российского пользователя — **Amvera Cloud** (или аналогичные сервисы, например, Railway/Render, но там сложнее с оплатой).

## Вариант: Запуск на Amvera Cloud

1.  **Регистрация:**
    *   Зайдите на [amvera.ru](https://amvera.ru/) и зарегистрируйтесь. Это российский облачный провайдер, удобный для Node.js приложений.

2.  **Создание проекта:**
    *   Нажмите "Создать проект".
    *   Выберите тип проекта: **Приложение**.
    *   Дайте ему имя (например, `credit-checklist-bot`).
    *   Выберите тариф (для начала подойдет самый дешевый или пробный, если есть).

3.  **Загрузка кода:**
    *   В настройках проекта найдите способ загрузки (можно через сайт перетаскиванием файлов или через Git).
    *   Если через сайт: Заархивируйте все файлы из папки `timur max` **КРОМЕ папки `node_modules`** в ZIP-архив.
    *   Загрузите этот ZIP-архив на Amvera.
    *   Система сама распакует, увидит файл `package.json` и поймет, что это Node.js проект.
    *   Она запустит команду `npm install` (установит зависимости) и затем `npm start` (запустит бота).

4.  **Проверка:**
    *   В логах на сайте Amvera вы должны увидеть строчку `Bot started...`.
    *   Бот теперь работает в облаке!

## Важные файлы

*   `package.json`: Содержит информацию о проекте и команде запуска (`npm start`).
*   `bot.js`: Основной код бота.
*   `.gitignore`: Список файлов, которые **не нужно** загружать на сервер (например, тяжелую папку `node_modules` и тесты).

## Если что-то пойдет не так

*   Проверьте логи (журнал событий) на сайте хостинга. Там будут видны ошибки, если бот упадет.
*   Убедитесь, что токен в `bot.js` работает и не менялся (хотя лучше хранить его в переменных окружения, но для начала так сойдет).

---

**Альтернатива: Запуск на своем компьютере как службы (Windows)**

Если вы хотите, чтобы бот работал на вашем ПК всегда (но ПК должен быть включен):

1.  Установите утилиту **PM2**:
    ```bash
    npm install pm2 -g
    ```
2.  Запустите бота через PM2:
    ```bash
    pm2 start bot.js --name "checklist-bot"
    ```
3.  Чтобы сохранить автозапуск при перезагрузке:
    ```bash
    pm2 startup
    pm2 save
    ```

Этот способ бесплатный, но требует включенного компьютера и интернета.
